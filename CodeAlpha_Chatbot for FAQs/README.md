# 🤖 Chatbot for FAQs (Phones & Laptops)

A simple AI-powered chatbot built using **Flask** and **Sentence Transformers** that answers questions about **Phones and Laptops**.

---

## 🚀 Features

* Semantic search (understands similar questions)
* Supports multiple products (phones & laptops)
* Handles short queries like *price, battery, gaming*
* Context-aware (separates phone & laptop queries)

---

## 🛠️ Tech Stack

* Python, Flask
* Sentence Transformers (NLP)
* HTML, CSS

---

## ▶️ How to Run

```bash
* Create Virtual Environment 
-------------------
python -m venv venv

.\venv\Scripts\activate

* Installation is mandatory
------------------------
pip install flask sentence-transformers

python app.py
```

Open: http://127.0.0.1:5000/
